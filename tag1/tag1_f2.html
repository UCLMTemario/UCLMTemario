Enterprise Resource Planning

Visión por proceso

Proceso de negocio

Conjunto estructurado y medible de actividades que se desarrollan en una organización con el objetivo de conseguir un resultado concreto para algún cliente o mercado específico. * Agrupan empleados de diferentes departamentos para completar el trabajo.

Proceso de negocio ejecutable

Proceso de negocio cuyas tareas pueden ser orquestadas en una plataforma software, usando tareas automatizadas y de personas.

ERP

Aplicación software construida alrededor de una base de datos y que soporta la mayor parte de las áreas funcionales y las actividades de negocio de una compañía, independientes de su ubicación.

Objetivo ERP

Detalles ERP

Carácteristicas ERP

  • Capacidad de parametrización: adapta el funcionamiento del sistema a las necesidades de la empresa, incorporando módulos a medida que la empresa lo requiere, sin necesidades a medida del cliente.
  • Interfaz de usuario avanzada y flexible: incorpora las últimas tecnologías y avances en la interfaz de usuario, con diversos dispositivos de acceso, diseño y desarrollo personalizados y acceso mediante web.
  • Integración con otras aplicaciones: facilita la comunicación y el intercambio de datos por medio de interfaces estandarizadas.

Estructura ERP

Adoptan una estructura modular con mayor coherencia e integración de los datos. Los módulos están interconectados y comparten una base común.
